4. Kontekstalne klase

Formatiraju tekst u odnosu na njegovo značenje:

  1. .text-muted - This text is muted.
  2. .text-primary - This text is important.
  3. .text-success - This text indicates success.
  4. .text-info - This text represents some information.
  5. .text-warning - This text represents a warning.
  6. .text-danger - This text represents danger.

Kontekstualne klase za boju pozadine:

  1. .bg-primary - This text is important.
  2. .bg-success - This text indicates success.
  3. .bg-info - This text represents some information.
  4. .bg-warning - This text represents a warning.
  5. .bg-danger - This text represents danger.
